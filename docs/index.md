## Kurzübersicht

- **Framework**: Next.js 16 (App Router)
- **Sprache**: TypeScript (strict mode)
- **Datenbank**: PostgreSQL mit Drizzle ORM
- **Authentifizierung**: Better Auth
- **UI**: shadcn/ui + Tailwind CSS 4
