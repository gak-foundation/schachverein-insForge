## Änderung 3: Sidebar mit RBAC-Filterung

**DATEI:** `src/components/layout/sidebar.tsx` — ÄNDERN

- Props: `{ role: string; permissions: string[] }`
- Import: `hasPermission`, `PERMISSIONS` from `@/lib/auth/permissions`
- Jedes Nav-Item bekommt `permission: Permission | null`
- Filter: `!item.permission || hasPermission(role, permissions, item.permission)`
- Leere Gruppen ausblenden

Permissions-Zuordnung:
| Item | Permission |
|------|-----------|
| Dashboard | null (immer sichtbar) |
| Mitglieder | MEMBERS_READ |
| Kalender | EVENTS_READ |
| Mannschaften | TEAMS_READ |
| Turniere | TOURNAMENTS_READ |
| Partien | GAMES_READ |
| Finanzen | FINANCE_READ |
| Protokolle | ADMIN_AUDIT |
