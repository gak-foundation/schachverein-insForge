## Technische Hinweise

### Datenbank
Schema bereits vollständig in `src/lib/db/schema.ts`. Bei Änderungen:
```bash
npm run db:generate
npm run db:push
```

### Auth
Bereits implementiert. Für geschützte Routes:
```typescript
const session = await auth();
if (!session) redirect("/login");
```

### RBAC
Permission-Check in Components:
```typescript
const hasAccess = hasPermission(userRole, userPermissions, PERMISSIONS.MEMBERS_WRITE);
```

### Validierung
Zod-Schemas in `src/lib/validations/index.ts` ergänzen.
