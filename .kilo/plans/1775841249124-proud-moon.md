## Architekturentscheidungen (festgehalten)

| Entscheidung | Begründung |
|---|---|
| Next.js App Router + Server Actions | Fullstack-Monolith, schnell iterierbar |
| Drizzle ORM statt Prisma | Bereits im Einsatz, näher an SQL, performant |
| Auth.js v5 (JWT) | Bereits implementiert, erweiterbar |
| shadcn/ui | Bereits installiert, konsistente UI |
| Zustand | Bereits installiert für Client-State |
| chess.js + react-chessboard | Bereits installiert |
| BullMQ + Redis | Bereits in package.json, für asynchrone Jobs |
| Swiss Pairing: extern (bbpPairings/JaVaFo) | Kein Self-Build, TRF als Austauschformat |
| PWA statt Native App | Geringerer Aufwand, Offline-fähig |

## DB-Erweiterungen (neue Tabellen)

```sql
-- DWZ/Historie
CREATE TABLE dwz_history (
  id UUID PRIMARY KEY,
  member_id UUID REFERENCES members(id),
  dwz INTEGER NOT NULL,
  source VARCHAR(50), -- 'manual', 'dewis_import', 'tournament'
  recorded_at DATE NOT NULL,
  created_at TIMESTAMP DEFAULT NOW()
);

-- Spielerverfügbarkeit
CREATE TABLE availability (
  id UUID PRIMARY KEY,
  member_id UUID REFERENCES members(id),
  date DATE NOT NULL,
  status VARCHAR(20) NOT NULL, -- 'available', 'unavailable', 'maybe'
  note TEXT,
  created_at TIMESTAMP DEFAULT NOW()
);

-- Dokumente
CREATE TABLE documents (
  id UUID PRIMARY KEY,
  title VARCHAR(300) NOT NULL,
  file_name VARCHAR(500) NOT NULL,
  file_url TEXT NOT NULL,
  category VARCHAR(50) NOT NULL, -- 'statute', 'protocol', 'certificate', 'other'
  mime_type VARCHAR(100),
  file_size INTEGER,
  uploaded_by UUID REFERENCES members(id),
  created_at TIMESTAMP DEFAULT NOW()
);

-- Newsletter
CREATE TABLE newsletters (
  id UUID PRIMARY KEY,
  subject VARCHAR(300) NOT NULL,
  body TEXT NOT NULL,
  sent_at TIMESTAMP,
  sent_by UUID REFERENCES members(id),
  created_at TIMESTAMP DEFAULT NOW()
);

-- Beitragssätze
CREATE TABLE contribution_rates (
  id UUID PRIMARY KEY,
  name VARCHAR(100) NOT NULL, -- 'Erwachsene', 'Jugend', 'Familie', 'Ehrenmitglied'
  amount DECIMAL(10,2) NOT NULL,
  frequency VARCHAR(20) NOT NULL, -- 'yearly', 'quarterly', 'monthly'
  description TEXT,
  valid_from DATE,
  valid_until DATE,
  created_at TIMESTAMP DEFAULT NOW()
);

-- Mannschaftskämpfe (separat von tournaments)
CREATE TABLE matches (
  id UUID PRIMARY KEY,
  season_id UUID REFERENCES seasons(id),
  tournament_id UUID REFERENCES tournaments(id),
  home_team_id UUID REFERENCES teams(id),
  away_team_id UUID REFERENCES teams(id),
  match_date DATE,
  location VARCHAR(300),
  home_score DECIMAL(5,1),
  away_score DECIMAL(5,1),
  status VARCHAR(20) DEFAULT 'scheduled',
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);

-- Einzelergebnisse je Brett
CREATE TABLE match_results (
  id UUID PRIMARY KEY,
  match_id UUID REFERENCES matches(id),
  game_id UUID REFERENCES games(id),
  board_number INTEGER NOT NULL,
  home_player_id UUID REFERENCES members(id),
  away_player_id UUID REFERENCES members(id),
  result VARCHAR(10),
  created_at TIMESTAMP DEFAULT NOW()
);
```
