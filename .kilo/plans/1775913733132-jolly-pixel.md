# Schachverein - Iterative Verbesserungen

## Überblick
Dieser Plan fokussiert sich auf kleine, pragmatische Verbesserungen an mehreren Bereichen des Schachverein-Management-Systems.

## 1. Dashboard-Verbesserungen (30 Min)

### 1.1 Dashboard Stats erweitern
- [ ] DWZ-Durchschnitt der aktiven Mitglieder anzeigen
- [ ] Anzahl Spiele diesen Monat
- [ ] Nächste Mannschaftskämpfe anzeigen
- [ ] Quick-Actions für häufige Aktionen (z.B. "Mitglied anlegen", "Turnier erstellen")

### 1.2 Dashboard UI verfeinern
- [ ] Stat-Cards mit Icons versehen
- [ ] Hover-Effekte verbessern
- [ ] Leere Zustände besser gestalten

## 2. Tabellen-Verbesserungen (45 Min)

### 2.1 Sortier-Funktion
- [ ] Sortierbare Tabellen-Header für alle Listenseiten
- [ ] URL-basierte Sortierung (behält Sortierung bei Refresh)
- [ ] Visuelle Indikatoren für Sortierrichtung

### 2.2 Paginierung
- [ ] Basis-Paginierung für Mitglieder (10/25/50 pro Seite)
- [ ] Paginierung für Turniere und Spiele

### 2.3 Leere Zustände
- [ ] Bessere Empty-States mit Icons und CTA-Buttons
- [ ] Konsistente Empty-State-Komponente erstellen

## 3. Detailseiten erweitern (45 Min)

### 3.1 Turnier-Detailseite
- [ ] Teilnehmerliste mit Rang, Punkten, Buchholz
- [ ] Rundenübersicht mit Paarungen
- [ ] Button "Teilnehmer hinzufügen"
- [ ] Turnier-Info-Karte (Datum, Ort, Zeitkontrolle)

### 3.2 Mannschafts-Detailseite
- [ ] Spielerliste mit Brett-Nummern
- [ ] Aufstellung anzeigen/bearbeiten
- [ ] Mannschaftskämpfe der Saison
- [ ] Kapitän anzeigen

## 4. UX-Verbesserungen (30 Min)

### 4.1 Navigation
- [ ] Breadcrumbs für tiefere Seiten
- [ ] Aktiver Nav-Punkt besser hervorheben

### 4.2 Feedback
- [ ] Toast-Benachrichtigungen für Actions
- [ ] Loading-States für Buttons

### 4.3 Forms
- [ ] Form-Validierung verbessern (Client-seitig)
- [ ] Bessere Error-Messages

## Reihenfolge

1. Dashboard-Stats erweitern (schneller Win)
2. Leere Zustände verbessern (betrift alle Seiten)
3. Turnier-Detailseite (wichtig für Schach-Workflow)
4. Tabellen-Sortierung (oft benötigt)
5. Kleinere UX-Verbesserungen

## Nicht in diesem Scope
- PGN-Viewer/Schachbrett (größeres Feature)
- E-Mail-Versand (Benachrichtigungen)
- DWZ-Sync mit externer API
- Mobile-Optimierung (ist bereits OK)
- PDF-Export

## Zeitaufwand geschätzt
- 2.5 Stunden insgesamt
- Fokus auf kleine, sichtbare Verbesserungen
